# Discord.Addons.InteractiveCommands
An addon for [Discord.Net 1.0](https://github.com/RogueException/Discord.Net). 

Allows you to use discord.py-esque methods to interact with the user. For example, waiting for user response, or automatically deleting a message after a few seconds.
